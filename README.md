## 🚀 Getting Started  

### Prerequisites  

- Install [Flutter SDK](https://flutter.dev/docs/get-started/install)  
- Setup an IDE (Android Studio / VS Code)  

### Installation  

1. **Clone the repository:**  
   ```bash
   git clone https://github.com/VickyDev810/DemoChat.git
   ```

2. **Install dependencies:**  
   ```bash
   flutter pub get
   ```

3. **Run the app:**  
   ```bash
   flutter run
   ```
